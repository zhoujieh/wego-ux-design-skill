使用说明

  • size=32/28/24 为可交互标签，支持点击切换 selected 态，20 为纯展示标签 (cursor: default)
  • grey 为默认灰色背景，white 为白色带边框，green 为绿色边框语义标签
  • selected 态感知：灰色/白色选中变绿色实底 + 绿色文字，绿色选中变绿色实底
  • 图标使用 wego-iconfont-s 字体图标（icon-jia / icon-cha / icon-youjiantou）
  • TagGroup 使用 flex-wrap + gap 布局，自动换行，间距统一为 var(--wg-spacing-8)
  • active 态需点击体验，focus-visible 态需 Tab 键聚焦查看
  • 所有样式值通过 var(--wg-*) 引用，零硬编码
