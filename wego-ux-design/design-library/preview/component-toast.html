使用说明

  • Toast 定位为页面底部居中 fixed，预览中以静态方式示意
  • data-state="hidden" 时隐藏（opacity: 0 + translateY）
  • action :active 效果需实际点击体验
  • action :disabled 时显示禁用态
  • 所有样式值通过 var(--wg-*) 引用，零硬编码
