使用说明

  • Spinner 通过 CSS animation 旋转，无需 JavaScript
  • data-state="hidden" 可隐藏整个 loading 容器
  • Skeleton pulse 动画模拟内容加载中的占位效果
  • Image 占位使用 mask-image 渲染图标，避免额外请求
  • 所有样式值通过 var(--wg-*) 引用，零硬编码
