Input 组件预览

scope: card / full  |  type: text / number / multiline  |  state: default / filled / focused / error / disabled

card 形态

card + text（默认）— placeholder

card + text（filled）— 已输入

card + text（focused）— 聚焦态

card + number + unit — 32 高度 / 居中加粗 / 单位

元

card + multiline — textarea

card + text + error — 错误提示

格式不正确

card + disabled — 只读

full 形态

full + text — 底部分割线

full + text（filled）— 已输入

full + text（focused）— 分割线高亮

full + error — 错误提示

请输入正确格式

full + disabled — 不可编辑

使用说明

  • focus 效果需点击输入框查看（card: 边框变色 / full: 底部分割线变色）
  • card 形态为圆角卡片式，full 形态为满宽底部分割线式
  • multiline 使用 <textarea> 而非 <input>
  • 所有样式值通过 var(--wg-*) 引用，零硬编码
