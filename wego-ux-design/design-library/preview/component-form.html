使用说明

  • horizontal 模式：标签固定宽度，内容区弹性伸缩
  • vertical 模式：标签在上，操作区占满宽度
  • 错误提示通过 wg-form--error + wg-form__hint 展示
  • 禁用状态使用 wg-form--disabled + input disabled 属性
  • 选择器使用 wg-form__action--select 点击区域 + 箭头图标
  • FormGroup 配合 wg-form--divider 实现分割线
  • 所有样式值通过 var(--wg-*) 引用，零硬编码
