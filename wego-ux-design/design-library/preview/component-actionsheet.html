使用说明

  • 模拟面板使用 position: relative 展示，实际使用时为 position: fixed 全屏覆盖
  • :active 按下效果需实际点击体验，模拟面板中可触摸查看
  • focus-visible 效果需 Tab 键聚焦查看
  • select 模式通过 aria-selected="true/false" 控制选中态
  • 所有样式值通过 var(--wg-*) 引用，零硬编码（safe-area 固定尺寸为规范约定）
