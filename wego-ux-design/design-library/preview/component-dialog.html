使用说明

  • 演示中的 dialog overlay 使用 position: relative 模拟容器展示，实际使用为 position: fixed 全屏遮罩

  • 按钮 active 态需点击体验，focus-visible 态需 Tab 键聚焦查看
  • 所有样式值通过 var(--wg-*) 引用，零硬编码
