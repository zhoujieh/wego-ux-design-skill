Setting row

配置行：左侧标题 + 描述，右侧控件。三个 control 变体：select / button / select-with-icon。多行堆叠到 .ds-settingrow__panel 中；多个 panel 用 .ds-settingrow__group 分组。

通用 — Settings panel (full layout)
基础设置
主题 选择主题
语言 选择您喜欢的按钮标签和应用内其他文本的语言
偏好设置
Editor 设置 Editor 中字体、word-wrap、window 设置等
快捷键设置 对 IDE 中的各个操作的快捷键进行自定义设置
导入配置 导入 VS Code 或 Cursor 中的所有插件、设置、代码片段以及快捷键配置到 TRAE 中。请注意，导入后将覆盖 TRAE 当前配置，且不可恢复
本地链接的默认打开方式 点击终端中的本地链接时，是否自动使用内置浏览器打开
Markdown 文件的默认打开方式 Markdown 文件将默认用此方式打开
数据管理
浏览器数据 浏览器中的站点数据（如 Cookies、本地存储等）
Variants — control types
A · Select 下拉选择型控件，最常见
B · Button 跳转 / 触发动作；支持 danger 文字色变体
C · Select + Icon 下拉选择 + 旁边一个 icon 按钮（次级配置入口）
